### 升级建议

#### 从 0.x 升级到 1.0.0
这是一个完全重写的版本，建议：

1. **备份现有配置**
   ```bash
   cp .env .env.backup
   cp package.json package.json.backup
   ```

2. **全新安装**
   ```bash
   npm uninstall @chimech/mcp-server
   npm install -g @chimech/mcp-server@latest
   ```

3. **迁移配置**
   - 使用新的环境变量格式
   - 更新客户端配置文件
   - 重新配置API密钥

4. **验证安装**
   ```bash
   chimech-mcp health
   chimech-mcp status
   ```
